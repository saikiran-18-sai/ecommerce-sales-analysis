# E-commerce Sales Analysis

## Project Overview
This project analyzes an e-commerce sales dataset to understand sales patterns, product performance, and customer behavior.

The goal is to simulate real-world data analysis tasks similar to those performed by analysts in large e-commerce companies.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI / Tableau
